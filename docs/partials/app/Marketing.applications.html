Marketing:applications

Description

Widget that allows to pick an application from the list

Dependencies

  • ApplicationsService

Usage

as element:
<applications
       model="{Number}"
       item="{Object}"
       empty="{Boolean}"
       empty-text="{String}">
</applications>

Parameters

ParamTypeDetails
modelNumber

selected application ID

itemObject

selected application object

emptyBoolean

allow empty option

emptyTextString

text on empty option

Example

The below example shows 4 linked controls: applications, networks, clusters, devices

<div class="item">
<label>Application:</label>
<applications
ng-model="filter.appId"
item="appItem"
empty="1"></applications>
</div>

<div class="item">
<label>Network:</label>
<networks
ng-model="filter.netId"
empty="1"
item="netItem"
restrict-to-ids="appItem.net_ids"></networks>
</div>

<div class="item">
<label>Server:</label>
<clusters
ng-model="filter.clusterId"
empty="1"
items="netItem.clusters"
></clusters>
</div>

<div class="item" ng-if="netItem.support_different_devices">
<label>Device:</label>
<devices
ng-model="filter.deviceId"
empty="1"></devices>
</div>
